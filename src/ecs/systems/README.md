# ECS Systems

System modules here operate on ECS component stores:

- `physics.js`: shared axis movement and collision helpers for creature entities.
- `drops.js`: drop gravity, pickup, lifetime, and mesh sync.
- `animals.js`: passive animal spawn, wander AI, damage, save/network sync.
- `mobs.js`: hostile mob spawn, AI, attacks, fuse/explosion, sunburn, save/network sync.
